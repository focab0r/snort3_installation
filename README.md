# snort3_installation

Instalador automatico para Snort3

## Instalacion

```
$> cd ~/Downloads
$> git clone https://github.com/focab0r/snort3_installation.git
$> cd snort3_installation
$> ./script
```

Con ejecutar el ficher `script` vale. Por favor, como user normal, no como root. Ya te pedir√° los permisos durante la instalacion. Ah, y esta probado en Kali, ultima version.
